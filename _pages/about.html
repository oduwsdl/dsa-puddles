Dark and Stormy Archives - Puddles

Visualizations of storytelling with web archive collections.

The Dark and Stormy Archives (DSA) project exists to provide storytelling solutions to improve the understanding of web archive collections. This DSA Puddles web site exists to feature the output of that project. Visit the Dark and Stormy Archives (DSA) project web site for information on research, software, and more that supports storytelling with web archives.
